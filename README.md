# App BodyCorp
📖 Sobre a BodyCorp

A BodyCorp é uma empresa fictícia que desenvolve tecnologias para melhorar a segurança e qualidade de vida de trabalhadores da indústria. Nosso foco é criar produtos eficientes, modernos e acessíveis.

🛠️ Tecnologias Utilizadas

ASP.NET Core MVC

C#

Bootstrap 5

HTML / CSS

Entity Framework Core

SQL Server (para o sistema de gestão)

🌐 1. Site Institucional

O site apresenta:

Navbar moderna com efeito blur

Seção inicial com texto de impacto

Tecnologias e diferenciais da BodyCorp

Cards de produtos

Banner visual

Formulário de contato responsivo

Footer com telefone e email da empresa

O layout utiliza Bootstrap + CSS customizado para criar um visual limpo e responsivo.

🗂️ 2. Sistema Interno de Gestão

Além do site institucional, o projeto inclui um sistema administrativo, permitindo gerenciar dados como:

Modelos (Models)

Cliente

Produto

Compra

ItemCompra

O que o sistema faz?

Cadastra e lista clientes

Registra compras e seus itens

Calcula valores totais

Relaciona produtos, clientes e compras via Entity Framework

Garante organização e controle das operações internas da empresa
